## Featured Projects

| Project | Description | Tech Stack | Link |
|----------|-------------|------------|------|
| **Lookback Option Pricing Engine** | Pricing floating-strike lookback options under the Black-Scholes framework. Implemented a C++ pricing engine connected to Excel via VBA using a Dynamic Link Library (DLL). Focus on numerical stability, performance optimization, and financial modeling accuracy. | C++, VBA, Excel | [Repo](https://github.com/Martino3108/Look_Back_Pricing) |
| **Additive Bachelier Option Pricing** | Calibration and option pricing under the Additive Bachelier model during stressed market conditions. Includes volatility surface fitting, parameter estimation via Fourier methods, exotic option pricing, and hedging strategy analysis with performance comparison. | Python, MATLAB, Quantitative Finance, Fourier Methods | [Repo](https://github.com/marcodeluca-gdl/Additive-Bachelier-Option-Pricing) |
| **Volatility Under Stress – Financial Econometrics Project** | Empirical investigation of pandemic-induced volatility shifts using HAC/Newey–West inference, leverage effect analysis through GARCH, T-GARCH and E-GARCH models, and high-frequency volatility estimation with microstructure-noise-robust realized variance and intraday VaR backtesting. | Python, Econometrics, GARCH Models, High-Frequency Data | [Repo](https://github.com/marcodeluca-gdl/Volatility-Under-Stress) |
| **Market Anomaly Detector** | Financial time-series anomaly detection using machine learning techniques. The project explores unsupervised models to identify abnormal market behaviors and potential structural breaks in time series data. Includes data analysis, model training, and evaluation pipeline. | Python, PyTorch / Scikit-learn, Time Series Analysis | [Repo](https://github.com/marcodeluca-gdl/Market-Anomaly-Detector) |
| **Mobile Phone Operator – Machine Learning Project** | End-to-end supervised learning project for customer behavior prediction in the telecom industry. Includes data preprocessing, feature engineering, model comparison, hyperparameter tuning, and performance evaluation using classification metrics. | Python, Scikit-learn, Pandas, Matplotlib | [Repo](https://github.com/marcodeluca-gdl/machine-learning-mobile-phone-operator) |
